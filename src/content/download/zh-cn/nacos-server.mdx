# Nacos Server 下载

:::tip

Nacos Server **3.1.0** 已正式发布。支持导入MCP服务和工具，同步最新官方MCP注册中心协议；支持A2A 注册中心功能，欢迎使用。

## 系统要求

:::note

以下是部署Nacos Server的最小系统要求。如果您的环境无法满足系统最小要求，可能会导致无法部署和启动Nacos Server。

:::

|             |                                                                                                                         |
|-------------|------------------------------------------------------------------------------------------------------------------------------------|
| **JDK/JRE** | 17及以上  |
| **CPU**  | 1核及以上，支持64位CPU    |
| **内存** | 2G及以上 |
| **硬盘** | 无最小要求，根据保留日志自行调整|
| **操作系统** | Linux, Mac OS X, Windows |
