# Nacos Server Download
:::tip
Nacos Server **3.0.1** has been officially released. Support MCP's multi-namespace management, MCP multi-version management and support MCP Registry API standard.

Feel free to [download and try it out](#stable-versions).

:::
## System Requirements
:::note
The following are the minimum system requirements for deploying Nacos Server. If your environment does not meet the minimum requirements, it may cause the Nacos Server to be unable to deploy and start.
:::
|             |                                                                                                                         |
|-------------|------------------------------------------------------------------------------------------------------------------------------------|
| **JDK/JRE** | 8 and above  |
| **CPU**  | 1 core or higher, 64-bit CPU support    |
| **Memory** | 2G or more |
| **Hard disk** | No minimum requirement, adjust according to log retention |
| **Operating system** | Linux, Mac OS X, Windows |
